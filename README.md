# WTWR â€” Weather-To-Wear Recommendation App

WTWR (Weather-To-Wear Recommendation) is a React-based frontend application that provides clothing recommendations based on real-time weather conditions. The project showcases practical frontend engineering skills, incremental feature development, and clean component architecture.

Repository: [https://github.com/Yournd/se_project_react_WTWR](https://github.com/Yournd/se_project_react_WTWR)

---

## Why This Project

This application was built to demonstrate real-world frontend development practices, including component-driven design, responsive UI implementation, and iterative feature delivery using version control.

---

## Key Engineering Highlights

- Designed and implemented a modular React component architecture
- Built weather-driven filtering logic to dynamically surface relevant clothing items
- Developed reusable modal components for item creation and preview workflows
- Implemented responsive layouts and mobile-specific UI enhancements
- Maintained clean commit history reflecting incremental, production-style development

---

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS
- External Weather API

---

## Feature Summary

- Weather-based outfit recommendations
- Temperature-aware clothing filtering
- Interactive modals for adding and viewing garments
- Responsive desktop and mobile layouts

---

## Development Approach

The project was developed incrementally using small, focused commits:

- Initial scaffolding with React and Vite
- Core layout and weather display components
- Data-driven clothing filtering logic
- UI/UX enhancements through modals and interactions
- Final mobile optimization and responsive refinements

## Local Setup

- git clone this repo
- cd se_project_react_WTWR
- npm install
- obtain OpenWeatherMap API key
- create a .env file with VITE_API_KEY=YOUR_KEY
- npm run dev
